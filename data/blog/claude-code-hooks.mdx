Claude Code에는 **Hook** 이라는 기능이 생겼습니다.

Hook 이벤트는 <b>`.claude/settings.json`</b> 파일에서 설정할 수 있습니다. `.claude` 폴더는 적용 범위(사용자, 프로젝트, 로컬)에 따라 위치가 각각 다르므로, 원하는 범위의 `settings.json` 파일을 찾아 설정하면 됩니다.

---

## 🔧 Hook 구조

```json
{
  "hooks": {
    "이벤트 이름": [
      {
        "matcher": "툴패턴",
        "hooks": [
          {
            "type": "command",
            "command": "실행할 명령어"
          }
        ]
      }
    ]
  }
}
```

### Hook Events

| 이벤트               | 실행 시점                                                              |
| -------------------- | ---------------------------------------------------------------------- |
| **PreToolUse**       | 작업 도구 호출 전                                                      |
| **PostToolUse**      | 작업 도구가 성공적으로 완료되면 즉시                                   |
| **Notification**     | 도구 승인 필요 시, 또는 입력 대기가 60초 이상 유휴 상태일 때           |
| **UserPromptSubmit** | 사용자가 프롬프트를 제출하면 Claude가 처리하기 전                      |
| **Stop**             | 모든 작업이 완료되면 실행 (사용자가 중단한 경우에는 실행되지 않음)     |
| **SubagentStop**     | 하위 에이전트 작업이 완료되면                                          |
| **PreCompact**       | 압축 작업 실행 전                                                      |
| **SessionStart**     | 새 세션 시작 또는 기존 세션 재개 시 (`/clear` 명령어 실행 시에도 실행) |
| **SessionEnd**       | 세션이 종료될 때                                                       |

### Matcher 구성

`matcher`는 도구 이름과 일치하는 패턴으로, 대소문자를 구분합니다. `PreToolUse` 및 `PostToolUse` 이벤트에만 적용됩니다.

매칭 가능한 주요 도구:

- `Task` — 하위 에이전트 작업
- `Bash` — 셸 명령어
- `Glob` — 파일 패턴 매칭
- `Grep` — 콘텐츠 검색
- `Read` — 파일 읽기
- `Edit`, `MultiEdit` — 파일 편집
- `Write` — 파일 쓰기
- `WebFetch`, `WebSearch` — 웹 작업

## 🔔 알림 설정하기

여러 이벤트 중 <b>`Notification`</b>과 <b>`Stop`</b>에 알림이 있으면 좋겠다고 생각했습니다.

- `Notification`: 허가가 필요한 작업 시 알림 (파일 CRUD 관련 접근 권한 확인 등)
- `Stop`: Claude가 모든 작업을 완료하고 정지할 때 한 번만 실행됨

### 알림 hook 코드

```json
{
  "hooks": {
    "Notification": [
      {
        "hooks": [
          {
            "type": "command",
            "command": "osascript -e 'display notification \"승인이 필요합니다.\" with title \"Claude Code\" sound name \"Ping\"'"
          }
        ]
      }
    ],
    "Stop": [
      {
        "hooks": [
          {
            "type": "command",
            "command": "osascript -e 'display notification \"모든 작업이 완료되었습니다\" with title \"Claude Code\" sound name \"Submarine\"'"
          }
        ]
      }
    ]
  }
}
```

- `osascript`: Mac에서 알림을 전송하는 명령어
- `display notification`: 상단 알림 목록에 표시될 메시지 설정
- `sound name`: 알림 소리 (Mac 기준 Sound 폴더의 파일명으로 지정)
