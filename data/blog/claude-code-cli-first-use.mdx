Anthropic에서 출시한 **Claude Code**는 터미널 기반의 AI 코딩 에이전트입니다. 기존 Claude 웹이나 Desktop 앱과 달리, 파일을 직접 읽고 수정하고 빌드까지 실행하는 방식으로 동작합니다.

"터미널에서 AI가 코드를 직접 건드린다"는 게 신기해서 바로 \$5 충전하고 테스트를 시작했습니다.

---

## 💰 사용 요금

- **Claude Pro/Max 구독** (\$20/월): 사용 포함
- **크레딧 충전**: \$5 단위로 충전 후 사용량만큼 차감
  - 참고: 테스트 시 \$5 충전 (결제 수수료로 \$0.5가 추가 결제됐다. 총 \$5.5)

---

## ⚡ 주요 단축키 (Quick Start)

VS Code와 함께 사용할 때 알아두면 편한 기본 기능들입니다.

- **`Cmd+Esc`**: Claude Code 실행
- **파일 diff 직접 적용**: 에디터에서 바로 변경사항 확인 및 적용
- **`Cmd+Option+K`**: 파일 참조 삽입

---

## 🛠️ 직접 사용해보기

### 1. 프로젝트 초기화 가이드 만들기

Claude 공식에서 권장하는 **프로젝트 초기화** 방식으로 첫 번째 테스트를 해봤습니다.

Claude Code가 프로젝트 전체 구조를 분석하고 `CLAUDE.md` 파일을 자동으로 생성해 줍니다. 이 파일은 이후 대화에서 Claude가 프로젝트 컨텍스트를 이해하는 기반이 됩니다.

비용: **\$0.66 달러 차감** (프로젝트 파일 전체를 분석하기 때문에 상대적으로 많이 소모됩니다)

### 2. VS Code에서 바로 사용하기

별도 설정 없이 **VS Code 내장 터미널에서 바로 사용** 가능합니다. `claude` 명령어를 실행하면 현재 열려 있는 프로젝트 컨텍스트가 자동으로 적용됩니다.

### 📝 개인 블로그 만들기

취업 전에 Next.js + TailwindCSS 기반 개발 블로그를 만들려고 템플릿을 다운받아 뒀는데, 2년 동안 방치해서 패키지 버전이 많이 낡아있는 상태였습니다. (~~마침 테스트하기 좋은 레포~~)

#### 1. 패키지 업데이트

**명령 프롬프트**

> next version update를 14에서 15버전으로 업데이트 하고 싶습니다. 먼저 현재 프로젝트의 Next.js 버전을 확인하고, 사용 중인 관련 패키지들을 정리해야 합니다.

Claude Code가 `package.json`을 확인하고 연관 패키지 버전을 정리한 뒤 업데이트를 진행했습니다.

가장 인상적이었던 점은 <b>스스로 빌드해보면서 오류를 반복적으로 수정하는 방식</b>이었습니다. '빌드 → 오류 확인 → 해결'을 반복하며, 처음에 의존성 버전 오류를 해결한 뒤 빌드 후 발생한 타입 오류까지 알아서 잡아줬습니다.

AI가 시킨 것보다 한 단계 앞서 수행하는 느낌이었습니다.

#### 2. 블로그 스타일 변경

**명령 프롬프트**

> 이제 내 개인 블로그 스타일로 변경할 거야. 어떤 걸 변경하면 좋을까.

Claude Code가 먼저 변경 가능한 항목들을 제안해 줬습니다. 이후 좀 더 구체적으로 요청해봤습니다.

**명령 프롬프트**

> Toss UI 스타일로 변경 해줘.

**Toss UI 스타일을 정확히 알고 있었습니다.** 요청 즉시 스타일이 반영됐습니다.

## 💻 사용 후기

Claude Code는 단순한 코드 제안 도구가 아니라, <b>터미널에서 직접 작업을 수행하는 AI 에이전트</b>입니다.

- 코드를 직접 수정하지 않아도 됨
- 빌드 오류를 스스로 반복하며 수정
- 프로젝트 전체 컨텍스트를 파악하고 작업
- Toss 등 알려진 UI 스타일 패턴도 학습되어 있음

다만 작업 하나에 \$0.66씩 차감되는 비용은, Pro 구독 없이 크레딧만으로 사용하면 생각보다 빠르게 소진될 수 있습니다. 무거운 작업은 Pro/Max 버전 활용을 추천합니다.
