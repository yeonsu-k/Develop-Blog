Notion API를 통한 페이지 관리 및 데이터베이스 연동으로 문서 작업을 자동화할 수 있습니다.

이 글에서는 Claude와 Notion을 MCP(Model Context Protocol)로 연결하는 방법과 실제 사용 경험을 공유합니다.

---

## ⚙️ 설정 방법

### 1. Claude Desktop App 설치

1. Claude 웹사이트의 [다운로드 페이지](https://claude.ai/download)에서 운영 체제에 맞는 데스크톱 애플리케이션을 다운로드
2. 설치 후 Claude Pro 계정으로 로그인 (MCP 사용 필수 요구사항)

### 2. Notion Integration 생성

1. [Notion Developers](https://developers.notion.com/) 페이지 접속
2. "New integration" 버튼 클릭하여 새 통합 생성
3. Integration 이름 설정 (예: "Claude MCP")
4. 생성된 Internal Integration Token (`secret_xxx...`) 복사 및 안전하게 보관

### 3. Claude Desktop Config 설정

**macOS:** `~/Library/Application Support/Claude/claude_desktop_config.json`

**Windows:** `%APPDATA%\Claude\claude_desktop_config.json`

```json
{
  "mcpServers": {
    "notion": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-notion"],
      "env": {
        "NOTION_API_KEY": "secret_your_token_here"
      }
    }
  }
}
```

### 4. 환경 변수 설정

**macOS/Linux:**

```bash
# ~/.zshrc 또는 ~/.bashrc에 추가
export NOTION_API_KEY="secret_your_token_here"
```

### 5. Notion 페이지/데이터베이스 권한 설정

1. 연동하려는 Notion 페이지 또는 데이터베이스를 열기
2. 우측 상단 "Share" 버튼 클릭
3. "Add connections" 서비스에서 생성한 Integration 선택 및 초대
4. Claude Desktop App 재시작

> 💡 **중요:** Notion에서 MCP 연결할 페이지를 직접 지정해야 해당 페이지에 대한 조회/추가/수정/삭제가 가능합니다.

---

## 🛠️ 주요 기능

### ✅ 현재 지원 기능

- **페이지 조회** (`notion_retrieve_page`): 페이지 내용과 메타데이터 가져오기
- **블록 조회/수정** (`notion_retrieve_block`, `notion_update_block`): 개별 블록 관리
- **데이터베이스 쿼리** (`notion_query_database`): 필터링, 정렬 조건으로 데이터 검색
- **검색** (`notion_search`): 페이지/데이터베이스 제목으로 검색
- **블록 추가** (`notion_append_block_children`): 페이지에 새로운 콘텐츠 추가
- **댓글 관리** (`notion_create_comment`): 페이지에 댓글 작성

## 📝 사용 예시

- 페이지 조회

  ```bash
  Claude: "Study 페이지 내용을 보여줘"
  → notion_retrieve_page 실행
  ```

- 데이터베이스 검색

  ```bash
  Claude: "8snippet 데이터베이스에서 사이버가드 관련 항목들을 찾아줘"
  → notion_query_database 실행
  ```

- 콘텐츠 추가

  ```bash
  Claude: "이 페이지에 새로운 섹션을 추가해줘"
  → notion_append_block_children 실행
  ```

## 💻 사용 경험

Claude Desktop에 연결하면, 노션에 MCP 연결할 페이지를 지정해야 해당 페이지에 조회/추가/수정/삭제가 가능합니다.

**'연결된 노션페이지 제목을 확인해줘'** 라고 요청하면, 연결된 페이지를 조회하고 요약해줍니다.

하위 페이지 생성 및 내용 추가도 가능합니다. 실제로 이 블로그 시리즈의 작성 전에 노션의 페이지들도 Notion MCP를 활용하여 작업했습니다.
