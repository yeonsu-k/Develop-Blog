## 프로젝트 소개

Next.js 15와 Tailwind CSS를 활용하여 제작한 개인 기술 블로그입니다.

### 주요 기능

- MDX 기반 콘텐츠 관리
- 다크 모드 지원
- 태그 기반 글 분류
- KBar 검색 기능

### 기술 스택

- **프레임워크**: Next.js 15 (App Router)
- **스타일링**: Tailwind CSS
- **콘텐츠**: Contentlayer + MDX
- **배포**: Vercel
